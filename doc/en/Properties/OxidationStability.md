# Oxidation stability (OS)

$OS = -0.0384 * DU + 7.770$ ([Wang *et al.*, 2012])

## Biodiesel standards

| Biodiesel standard         | Limit    |
| -------------------------- | -------- |
| China (GB/T 20828-2007)    | [-;+∞]   |
| Europe (EN 14214-2005)     | [6.0;+∞] |
| Russia (ГОСТ Р 53605-2009) | [6.0;+∞] |
| USA (ASTM D6751-2003)      | [3.0;+∞] |
