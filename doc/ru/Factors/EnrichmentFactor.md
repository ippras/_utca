# Фактор обогащения

$$\frac{X_{\{2: i\}}}{3 X_{\{1,2,3: i\}}}$$

* **Описание**: Фактор обогащения (EF) измеряет степень обогащения данной жирной
  кислоты положения sn-2 триацилглицерина (TAG) по сравнению с ее общей
  концентрацией. Более высокий SF указывает на большее конкурентное преимущество
  конкретной жирной кислоты для этой позиции.
* **Интерпретация**:
  * `< 1.0` указывает на предпочтение позиций sn-1 и sn-3;
  * `= 1.0` указывает на отсутствие предпочтения какой-либо позиции;
  * `> 1.0` указывает на предпочтение позиции sn-2.
