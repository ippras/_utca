# Cetane number

Цетановое число (CN) — это способность топлива быстро воспламеняться после
впрыска, и более высокое значение указывает на лучшее качество воспламенения
топлива. Это один из наиболее важных параметров, учитываемых при выборе
метиловых эфиров для использования в качестве биодизельного топлива. Для хорошей
работы двигателя требуется адекватное цетановое число. Высокое цетановое число
помогает обеспечить хорошие характеристики холодного запуска и минимизировать
образование белого дыма.

## Стандарты биодизельного топлива

| Регион                                  | CN  |
| --------------------------------------- | --- |
| США (ASTM D6751-2003)                   | 47  |
| Европейских организаций (EN 14214-2005) | 51  |
| Китая (GB/T 20828-2007)                 | 49  |
